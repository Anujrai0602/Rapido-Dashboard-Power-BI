# Rapido-Indore-Power-BI-Dashboard

This project provides a deep dive into urban mobility trends in Indore, analyzing over 30,000+ ride records to evaluate service efficiency and revenue patterns. By processing raw data into an interactive dashboard, I identified critical demand hotspots—specifically across Geeta Bhavan, MR9, and Palasia—and quantified the dominance of Bike over Auto services in navigating city traffic. This analysis highlights a 94% ride completion rate and a ₹4.2M revenue flow, showcasing the balance between high-volume operations and service quality.
